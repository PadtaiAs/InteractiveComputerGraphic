รหัสนิสิต: 6721601613
หมู่: 700
การศึกษา: นักศึกษาภาควิชาวิทยาการคอมพิวเตอร์ คณะศิลปศาสตร์และวิทยาศาสตร์ มหาวิทยาลัยเกษตรศาสตร์
ทักษะ: Java, Assembly (RISC-V), SQL / Database, HTML/CSS/JavaScript, and Computer Graphics Algorithms

Project Title: 24-bit Sprite Low-Level Transformation System

โปรแกรมทำการจอง Buffer บนหน่วยความจำผ่านหน้าจอโครงสร้าง ctx.createImageData
เพื่อเข้าถึงและควบคุมค่าสีแต่ละเม็ดพิกเซลบน Canvas ได้โดยตรง
ในรูปแบบ Array ผืนผ้าใบเดี่ยว [R, G, B, A, ...]  การคำนวณสีระบบ 24-bit (True Color): ผ่านฟังก์ชัน putPixel(x, y, r, g, b)
ที่รับพิกัดแผงสกรีนและทำการแยกพิกเซลย่อยแบบช่องสีละ 8-bit (Red 8-bit, Green 8-bit, Blue 8-bit)
ทำให้สามารถประมวลผลการผสมสีที่มีความลึกของสีแบบ 24-bit ได้สูงสุดถึง 16.7 ล้านสีอย่างอิสระ

Scaling (การย่อ/ขยาย): นำพิกัดหลังหมุนไปคูณเข้ากับค่าอัตราส่วนการขยายพิกัด scaleX และ scaleY ตรง ๆ เพื่อขยายมิติพิกเซล
Translation (การเลื่อนตำแหน่ง): ขั้นตอนสุดท้ายนำพิกัดที่แปลงขนาดและทิศทางแล้วไปบวกเข้ากับพิกัด x และ y (พิกัดตามตำแหน่งเมาส์ MouseX, MouseY) เพื่อย้ายวัตถุไปแสดงผลบนพิกัดสกรีนปลายทางที่ต้องการ
Primitive Rendering: นำจุดยอดที่คำนวณได้ ไปวาดเติมเต็มพิกเซลด้วยฟังก์ชัน drawEllipse ส่งผลให้ Sprite ที่แสดงออกมาสามารถยืดหด หมุน และเคลื่อนที่ได้อย่างสมบูรณ์แบบ

Keyboard Interaction: ทำการผูก Event Listener กับปุ่มกดบนคีย์บอร์ดแบบ Real-time:
  ปุ่ม 1, 2, 3, e ใช้ในการสลับค่าชุดสีระบบ 24-bit RGB ของพิกเซลวัตถุ
  ปุ่มลูกศรซ้าย/ขวา (ArrowLeft/ArrowRight) ควบคุมการเพิ่ม-ลดมุมหมุน degree
  ปุ่มลูกศรอัป/ดาวน์ (ArrowUp/ArrowDown) ควบคุมการบวก-ลบอัตราการย่อขยายขนาดพิกเซล scaleX, scaleY (โดยจำกัดขอบเขตขั้นต่ำไว้ที่ 1 เพื่อป้องกัน Matrix พัง)  
Mouse Tracking Interaction: ดักจับพิกัดตัวชี้เมาส์ผ่านแผงหน้าจอ Canvas และทำการแปลงอัตราพิกัดจากขนาด CSS ไปเป็นพิกัดจริงบนราสเตอร์ Canvas ผ่านฟังก์ชัน toScaledMouseCoords ทำให้ตัวนำเข้า MouseX และ MouseY เลื่อนตำแหน่งตามเมาส์ได้ทันที
Animation Loop: ขับเคลื่อนการเรนเดอร์ภาพให้เป็นภาพเคลื่อนไหวลื่นไหลต่อเนื่องด้วยระบบวงรอบลูป requestAnimationFrame(animate) ซึ่งทำการล้างเฟรมภาพเก่าทุก ๆ เฟรมด้วยการเทพื้นสีดด้วย drawRectFilled ก่อนที่จะวาดเส้นแกนกราฟิกและอัปเดต Sprite ตัวใหม่ลงไปบนหน้าจอโฮสต์ในทันที

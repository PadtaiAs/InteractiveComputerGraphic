รายละเอียดโครงงาน (Project: 24-bit Sprite Low-Level Transformation System)

  • Memory Buffer & 24-bit Color: จอง Buffer บนหน่วยความจำผ่าน ctx.createImageData ควบคุมพิกเซลบน Canvas โดยตรงในรูปแบบ Array [R, G, B, A, ...] และคำนวณสี 24-bit (True Color) ผ่าน putPixel(x, y, r, g, b) แยกช่องสีละ 8-bit รองรับสูงสุด 16.7 ล้านสี
  • Scaling & Translation: คูณพิกัดหลังหมุนด้วย scaleX, scaleY เพื่อย่อ/ขยาย และนำไปบวกพิกัด MouseX, MouseY เพื่อเลื่อนตำแหน่งวัตถุ
  • Primitive Rendering: นำจุดยอดไปวาดเติมพิกเซลด้วยฟังก์ชัน drawEllipse ทำให้ Sprite ยืดหด หมุน และเคลื่อนที่ได้สมบูรณ์
  • Keyboard Interaction: สลับชุดสี 24-bit RGB ด้วยปุ่ม 1, 2, 3, e และควบคุมการหมุน/ย่อขยายด้วยปุ่มลูกศร (จำกัดขั้นต่ำที่ 1)
  • Mouse Tracking: แปลงพิกัด CSS เป็นพิกัดจริงบนราสเตอร์ Canvas ผ่าน toScaledMouseCoords ให้วัตถุเคลื่อนตามเมาส์ทันที
  • Animation Loop: ใช้ requestAnimationFrame(animate) ล้างเฟรมเก่าด้วย drawRectFilled ก่อนอัปเดต Sprite ใหม่ต่อเนื่อง
